Responsive Web Design

What is it?

Ever wonder how you can access the same webpage from your computer, ipad, iphone, etc? It's because Responsive web design forms to fit your screen.

How does it work?

Responsive web design uses HTML and CSS elements to include all information despite the platform it is viewed on. Using tools like: .

    Viewport: sets the visible content for the page

    Media Query: features such as width, height, and color

    Flex Box: As dimensions change, flex boxes rearrange themselves to best show content
